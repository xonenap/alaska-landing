## Features

- Responsive design (looks great on mobile & desktop)
- Dark-themed cards with Alaskan animals (grizzly bear, moose, bald eagle)
- Section about glaciers with a fun fact
- Bootstrap carousel for activities (kayaking, dog sledding, hiking)
- Northern Lights subscription form with blur effect
- Simple & clean layout
